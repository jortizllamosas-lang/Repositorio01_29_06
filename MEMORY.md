# MEMORY

Registro de información importante sobre el proyecto, indicada por el usuario.

## Estructura de archivos de reporte en CLIENTY

En cada archivo Excel ubicado dentro de las carpetas que están dentro de la carpeta `CLIENTY`:

- **Fila 1**: contiene el nombre del asesor comercial que envió la campaña, ubicado después de las palabras "Enviado por:".
- **Fila 2**: contiene el nombre de la campaña, ubicado después de las palabras "Texto enviado:".
- **Fila 3 en adelante**: tabla de 9 columnas, donde la fila 3 es el encabezado de cada columna.
